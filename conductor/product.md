# Product Guide: freelance-site (nickfig.dev)

## Product Vision
The professional freelance landing page and developer portfolio for Nick Figliolia. The site serves as a digital storefront to attract high-quality contract clients, showcase engineering excellence through detailed technical case studies, and provide a seamless channel for inquiry handling.

## Target Audience
- **Tech Founders & Startup CTOs:** Looking for premium frontend/full-stack freelance software development or custom dashboard integrations.
- **Product Managers & Business Owners:** Seeking independent developers to design, develop, and launch specialized applications or client portals.
- **Peers & Engineering Directors:** Reviewing technical competencies, engineering style, and system design philosophies.

## Core Value Proposition
A premium, hyper-performant, edge-rendered developer landing page demonstrating meticulous attention to detail. It features custom design variables, interactive flourishes, dynamic MDX-driven technical deep-dives, and an automated inquiry form, proving that Nick Figliolia delivers state-of-the-art web experiences.

## Key Features & Capabilities
1. **Dynamic Case Studies:** Managed as local Markdown/MDX files under `src/content/work`, rendering comprehensive descriptions, problem statements, solutions, metrics, and tech stacks.
2. **Interactive UI Flourishes:** Glassmorphism headers, responsive hover animations with colored glow shadows, smooth page scroll alignments, and button click haptics.
3. **Availability & Pricing Panel:** Instantly synced status display (Available/Booked) with maximum client intake indicators and baseline service pricing controlled via a central site config.
4. **Cloudflare Edge Deployment:** Designed specifically for `@cloudflare/next-on-pages` compilation, guaranteeing rapid page speed worldwide.
5. **Secure Form Delivery:** Integrates seamlessly with Web3Forms API to process inquiries without revealing server endpoints.
