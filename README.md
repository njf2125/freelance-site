# nickfig.dev

The professional portfolio and freelance landing page for **Nick Figliolia**. 

[**Visit nickfig.dev**](https://nickfig.dev)

## Overview

This site is a custom-built portfolio designed to showcase high-quality software engineering projects, technical case studies, and full-stack capabilities. It emphasizes clean typography, performance, and a direct, professional tone.

## Tech Stack

- **Framework:** [Next.js 14+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Content:** [MDX](https://mdxjs.com/) for case studies and documentation
- **Styling:** Vanilla CSS with modern CSS variables for theming
- **Forms:** Integration with Web3Forms for inquiry handling
- **Deployment:** [Cloudflare Pages](https://pages.cloudflare.com/)

## Key Features

- **Dynamic Case Studies:** Projects are managed via MDX files with custom frontmatter, allowing for rich, version-controlled technical writing.
- **Mobile First:** Fully responsive design that maintains high legibility across all devices.
- **Edge Performance:** Optimized for deployment on Cloudflare's edge network for near-instant load times globally.
- **Modern Architecture:** Uses the latest Next.js patterns, including React Server Components (RSC) and strict TypeScript.

## Local Development

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the site:**
   Navigate to [http://localhost:3000](http://localhost:3000)
